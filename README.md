# React基礎学習プロジェクト

このプロジェクトは、Reactの基本的な概念を学ぶためのサンプルアプリケーションです。

## 機能概要

シンプルなカウンターアプリケーションを通じて、以下のReactの基本概念を学ぶことができます：

- コンポーネント間の状態管理
- Props（親から子へのデータ受け渡し）
- コールバック関数（子から親への通知）
- 履歴管理による状態変更の可視化

## プロジェクト構成

```
src/
├── App.tsx (メインコンポーネント)
├── components/
    ├── Counter.tsx (カウンター機能)
    └── History.tsx (履歴表示機能)
```

### コンポーネントの役割

#### App.tsx（親コンポーネント）
- アプリケーション全体の状態管理
- ChakraUIのプロバイダーとレイアウト提供
- 管理する状態：
  - `count`: カウンターの現在値
  - `history`: 操作履歴（最新10件）

#### Counter.tsx
- カウンターのUI表示
- Props:
  - `count`: 現在の値
  - `onIncrement`: 増加時のコールバック
  - `onDecrement`: 減少時のコールバック
- ChakraUIのアイコンボタンを使用

#### History.tsx
- 操作履歴の表示
- Props:
  - `history`: 操作履歴の配列
- 最新の操作を強調表示
- スクロール可能なリスト表示

## データフロー

1. ユーザーがボタンをクリック
2. Counterコンポーネントから親へコールバック関数を通じて通知
3. Appコンポーネントで状態を更新
   - カウント値の更新
   - 履歴への追加
4. 更新された状態が子コンポーネントへ反映

## 技術スタック

- React + TypeScript
- ChakraUI（モダンなUIコンポーネント）
- Vite（開発環境）

## 特徴

- モダンなUI実装
- コンポーネントの責務が明確に分離
- TypeScriptによる型安全性
- 詳細なコメント付きコード
- 履歴機能による状態変更の可視化

## 学習ポイント

- Reactコンポーネントの基本構造
- Props、Stateの使い分け
- 親子コンポーネント間のデータフロー
- UIライブラリの活用方法
- TypeScriptを使用したReactの型定義
